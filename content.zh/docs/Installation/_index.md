# 安装说明

要使用Lomorage，您需要在本地网络运行Lomorage服务程序，您可以选择在Windows，macOS，Ubuntu，Raspbian, Armbain上运行，可以使用x86或者arm架构的机器。然后您需要去应用程序商店下载手机客户端应用。在运行了Lomorage服务程序之后，手机客户端会自动发现服务，无需配置，然后您就可以开始备份照片和视频了。
